Добрый день.
Создана БД factory с таблицами (employyes, position, vacation).
Таблица employyes содержит поля id_position и id_vacation, 
которые ссылаются на таблицы position, vacation.
Страница http://project/company.php содержит информацию
о сотрудниках, которые выбрали дату отпуска. 
Реализован поиск сорудников по дате отпуска в промежутке с ... по ... .
Страница http://project/list.php содержит инфоромацию о сотрудниках завода.
Столбец должность подгружается с таблицы position. 
Реализовано удаление сотрудника кнопкой DELETE.
Указана кнопка на переход страницу для создания сотрудника.
Страница http://project/add.php предлагает пользователю ввести информацию 
о новом сотруднике(должность предлагается выбрать из списка).
При нажатии кнопки Добавление сотрудник добавляется в форму http://project/list.php.
Реализация заняла 3 дня.
